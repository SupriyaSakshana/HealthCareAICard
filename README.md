# HealthCareAICard
This project is an AI-powered healthcare 
Hybrid Recommendation System that combines:
<img width="1332" height="307" alt="image" src="https://github.com/user-attachments/assets/acb9ca3f-3c1b-4056-8b6a-948ef13fdf8a" />




Content-Based Filtering (using TF-IDF and Cosine Similarity)

Collaborative Filtering (based on user ratings and interactions)

This hybrid approach helps in recommending suitable medicines and healthcare items based on both user preferences and similar user behavior.

The project is implemented as a web application using the Flask framework, providing a user-friendly interface for:

Registration and Login

Disease Prediction

Medicine Recommendations

Profile Management

Secure Session Handling

The system aims to assist users in understanding their health conditions and getting preliminary guidance before consulting medical professionals.
